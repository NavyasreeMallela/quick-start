# **Job Trend Analysis using Power BI**



#### 1\. Project Overview

This project focuses on analyzing job market trends using cleaned LinkedIn job posting data.

The main objective is to identify in-demand job roles, required skills, job locations, and salary trends.

The insights from this project help understand current hiring patterns in the job market.

##### 

---



#### 2\. Objectives

\- Analyze job postings data to identify hiring trends

\- Find the most in-demand job roles

\- Analyze required skills for different roles

\- Study job distribution across locations

\- Understand salary trends by role



---
